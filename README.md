# Bookie

### Installation
- Install Node
    - https://nodejs.org/en/download/
- Install Bower
    - npm install -g bower
- Install Gulp


####For Developer

1. git clone https://github.com/Bookie-TSP/Bookie-FrontEnd.git
2. Run 'npm install'
3. Run 'bower install'
4. Run 'gulp'

####For Production

1. git clone https://github.com/Bookie-TSP/Bookie-FrontEnd.git
2. Run 'npm install'
3. Run 'bower install'
3. Run 'node node-server.js'
