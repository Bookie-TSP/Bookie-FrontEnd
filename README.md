# Bookie

####For Developer
- Install node.js and gulp

1. git clone https://github.com/Bookie-TSP/Bookie-FrontEnd.git
2. Run 'npm install'
3. Run 'gulp'

####For Production 
- Install node.js and gulp

1. git clone https://github.com/Bookie-TSP/Bookie-FrontEnd.git
2. Run 'npm install'
3. Run 'node node-server.js'
